unspent transaction output (UTXO)

Phonetic: /ˌʌnˈspɛnt trænˈzækʃən ˈaʊtpʊt/, /ˌjuːˈtiːˈɛksoʊ/

Part of Speech: noun

Category:

Definition: An unspent transaction output (UTXO) represents some amount of digital currency which has been authorized by one account to be spent by another. UTXOs use cryptography.html">public key cryptography to identify and transfer ownership between holders of public/private key pairs. UTXOs are formatted with the recipient's public key, thus restricting the ability to spend that UTXO to the account that can prove ownership of the associated private key. The UTXO can only be spent if it includes the signature.html">digital signature associated with the public key attached the last time it was sent.
