soft fork

Phonetic: /sɒft fɔrk/

Part of Speech: noun

Category:

Definition: A change to the blockchain protocol resulting in only previously valid blocks and transactions being made invalid. Since old nodes will recognize the new blocks as valid, a soft fork is backward-compatible. However, this can result in a potential divide in the blockchain, as the old software generates blocks that read as invalid according to the new rules. Contrast with 'fork.html">hard fork'; for more basic information regarding forks, see 'fork'.
