bridge

Phonetic: /brɪdʒ/

Part of Speech: noun

Category:

Definition: A bridge is a tool built to move assets from one network to another. It’s also a verb, used to describe that action: “I bridged my ETH from Ethereum mainnet to Arbitrum.” Not all bridges are created equal, and you should be informed about what you’re doing before you use one..
